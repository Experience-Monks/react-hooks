# use-window-size

**Description**

Detects window size resize

**Props**

- Debounce (ms)

**Returns**

- { innerWidth, innerHeight }

```js
const debounce = 200;
const { innerWidth, innerHeight } = useWindowSize(debounce);
// returns window innerWidth and innerHeight
```

</Playground>

```jsx
import React from "react";
import { useWindowSize } from "@jam3/react-hooks";

const WindowSize = () => {
  const { innerWidth, innerHeight } = useWindowSize(100);

  return (
    <div>
      window width: {innerWidth}
      <br />
      window height: {innerHeight}
    </div>
  );
};
export default WindowSize;
```
